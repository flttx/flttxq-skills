# React Native 审查清单

适用于 React Native、Expo 项目。

## 重点检查

- 首屏链路、权限链路、导航链路是否破坏既有不变量。
- AppState、焦点切换、返回手势、前后台切换是否处理正确。
- 原生权限、文件、音视频、网络上传是否存在平台差异问题。
- 大列表、图片、录音、视频、动画是否存在性能或资源释放风险。
- iOS / Android 行为是否一致，是否遗漏平板或横竖屏场景。
- 构建配置、OTA、环境变量、渠道配置是否一致。

## 常见误区

- 只在一个平台验证，忽略双端差异。
- 组件层直接拼原生能力细节，缺少 service / hook 封装。
