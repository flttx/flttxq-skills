# Apple App Store (App Store Connect)

## Typical Submission Text Blocks

1. What's New
- Common limit in App Store Connect: up to 4000 characters.
- Keep user-facing and concise.

2. App Review Information / Notes
- Explain reviewer path from launch to core feature.
- If login required, provide valid review access path.
- For account deletion/cancellation capability, state in-app entry clearly.

3. Privacy and Compliance Alignment
- Ensure release notes, review notes, permission behavior, and privacy policy are consistent.
- If AI or third-party processing exists, disclose data type, recipient, and purpose consistently.

## App Store High-Risk Mismatch Items

1. Claiming capabilities that are not visible in current build.
2. Stating "no account needed" when login gates core path.
3. Missing deletion/cancellation path in notes when feature exists.
4. Privacy wording inconsistent with actual data transmission.
