## B. 审核备注模板（自注册口径）

```text
无需提供测试账号，审核员可自行注册体验。
审核路径：
1）启动 App，完成协议勾选；
2）进入注册并完成手机号+验证码；
3）验证本版本关键入口与流程；
4）完成核心功能体验。
说明：验证码仅用于身份核验，不读取短信内容或通讯录。
注意：请勿短时间频繁发送验证码，可能触发上游限流，请间隔后重试。
```

## C. 权限说明模板（音频类应用）

```text
1. 允许应用录制音频：用于用户主动录音上传与处理；仅在触发录音时申请。
2. 允许应用修改或删除存储卡中的内容：用于导入音频与上传过程临时缓存写入/清理。
```
